## Compile
It's a very easy to compile this bot
 * <b>You need to have a brain</b>
 * Install golang (google if you don't know how to)
 * Fill Bot Token, API Token and Reports Chat in `configurators.go`
 * Type `go build .`
 * Run (`./quotes`)
